Project: Web scraping

Tools used: Python, Jupyter notebook, Visual studio code

Scope

In this project I scrape data from the website of the romanian store SnowSports.

As a snowboarder, I’m interested in finding good deals on specific gear and I think this project showcases my technical skills with Python and data analysis tools. Also, this project demonstrates my abilities to:

  • extract information from websites (web scraping);
  • process and analyze the extracted data using libraries (data analysis);
  • set up a program to automatically check and save price information at regular intervals (automation).

    • Steps

    For the web scraping project I took the following steps:


    1. Imported the necessary libraries: 'BeautifulSoup', 'requests', 'time' and 'smtplib' for web scraping, data manipulation and e-mail functionality.

    2. Defined the 'check_price' function to:

  • download website content using specific headers for anonymity;
  • extract product title and price;
  • clean and formatt the extracted data (removing unnecessary characters and converting the price to a number);
  • save the extracted data (title, price and current date) to a CSV file;
  • send an e-mail notification if the price falls below the specific threshold.

    • 3. Set the target website URL by specifying the web address of the specific snowboard gear I'm interested in tracking.

    4. Changed the headers to mimic a regular browser and ensure smooth website interaction.

    5. Downloaded the website's content (just like a web browser).

    6. Processed the download through parsing the HTML content (using 'BeautifulSoup' for easier data extraction).

    7. Located and extracted product information from the website's code structure.

    8. Cleaned and formatted extracted data (removed extra spaces, unwanted characters and converted the price to a numerical format for further analysis).

    9. Saved data to the CSV file.

    10. Scheduled price checks and e-mail notifications by setting the program to run the 'check_price' function at regular intervals (every 12 hours) and send an e-mail if the price drops below a certain level.

    Inshights


    This project provides insights into price trends for the snowboard gear I’m interested in.

    By running this program over time, I can track price fluctuations and make informed purchase decisions.
